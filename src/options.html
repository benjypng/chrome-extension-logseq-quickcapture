Quick Capture Options (Logseq v0.8.12 onwards)

Send Quick Capture To:

Indicate where you want to send your quick capture item to.

  • today : Send to today's page
  • cursor : Send to wherever your cursor is
  • {page name} : If a valid page name is indicated, it will be sent to that specific page
Append to end of page?

This option is ignored if you select cursor in the above option.
